Dissertation Summary

Research Objective

Develop and evaluate machine learning models for detecting Command and Control (C2) communication patterns in network traffic using metadata-based features.

Key Research Questions

  • Which network features (e.g., flow-based, timing-based, protocol metadata) are most predictive for distinguishing C2 communication from legitimate traffic?
  • Can synthetic datasets generated in a lab environment provide reliable training data for real-world C2 detection?
  • Which machine learning models perform best in detecting C2 communication in synthetic network traffic?
  • Can lightweight ML models be deployed for real-time detection of C2 on low-resource edge devices or routers?

Methodology

Comparative analysis of Decision Tree, Random Forest, and SVM algorithms using synthetic data from controlled lab environments.

Expected Contributions

Enhanced detection capabilities for modern C2 communications with improved real-world applicability and reduced false positive rates.
