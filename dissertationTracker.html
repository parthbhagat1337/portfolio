Research Overview

Research Objective

Develop and evaluate lightweight machine learning models to detect Command-and-Control (C2) communications in encrypted network traffic. Focus on flow-based and timing-based features for real-time detection on resource-constrained devices.

Research Questions

  • Which network features best distinguish C2 from legitimate traffic?
  • Can synthetic datasets effectively train real-world C2 detectors?
  • Which ML models excel at encrypted C2 detection?

Research Methodology

Public Dataset and Synthetic dataset generated in a virtual lab simulation of C2 and legitimate traffic, feature extraction (flow patterns, timing, metadata), multiple ML model training and evaluation, with a focus on lightweight models for edge deployment.

Expected Output

  • Labeled synthetic C2 traffic dataset
  • Comparative ML model analysis for C2 detection
  • Analysis of a different C2 framework
  • Key feature identification for encrypted C2 traffic
